// META ANALYSIS (December 2024):
// - Izzet Lessons is #1 (14%) - relies on counterspells and graveyards
// - Dimir Midrange and Selesnya Landfall rising
// - The meta is very spell/instant-heavy
// - Almost no mainboard graveyard hate
// - Red removal dominates (punishes mana dorks... but not 4+ toughness dinos)
//
// WHY THIS DECK:
// 1. Cavern of Souls makes ALL dinosaurs uncounterable vs Izzet
// 2. Pugnacious Hammerskull (3/6) and Hulking Raptor (7/7) dodge red removal
// 3. Trumpeting Carnosaur provides Discovery value the meta forgot
// 4. We go OVER Dimir's value engines with raw size
// 5. Triumphant Chomp is premium removal nobody expects
// 6. Nobody is prepared for turn 3-4 massive dinosaurs
//
// ARCHETYPE: Gruul Dinosaur Ramp (Tier 3 / Rogue)
// ===================================================================
